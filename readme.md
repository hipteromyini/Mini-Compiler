# Mini-Compiler
## 简介 
项目设立于2020-11-5，实现迷你编译器功能，用于完成课程《编译原理课程设计》。

成员 ：吴树晖、柳泉旭、杨成玺、李博、崔志莹、周晴宇。
## 特点及功能
特点：带有代码高亮的文本编辑器,支持（待补），生成汇编中间代码，并在模拟的虚拟机中运行。

功能：

int类型的定义

for  while循环以及{}块代码

简单函数的定义

界符 //  ()  {} [] 

运算符 + - * / ^ & | && || >> << == != 
...
